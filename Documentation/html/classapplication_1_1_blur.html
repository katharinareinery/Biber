Detailed Description

Blurring (smoothing) is the commonly used image processing operation for reducing the image noise. The process removes high-frequency content, like edges, from the image and makes it smooth. In general blurring is achieved by convolving (each element of the image is added to its local neighbors, weighted by the kernel) the image through a low pass filter kernel.

src: The source (input image) for this operation. dst: The destination (output image) for this operation. sigmaColor: Filter sigma in the color space. sigmaSpace: Filter sigma in the coordinate space. Size( w, h ): Defines the size of the kernel to be used ( of width w pixels and height h pixels). Point(-1, -1): Indicates where the anchor point (the pixel evaluated) is located with respect to the neighborhood. If there is a negative value, then the center of the kernel is considered the anchor point.
